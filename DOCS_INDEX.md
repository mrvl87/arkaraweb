### Saat Error/Bug

2. Check [Troubleshooting section](./WALKTHROUGH.md#-troubleshooting)
3. Check [Known Issues](./README.md#-known-issues)
4. Search Google dengan error message

1. Review dokumentasi yang relevan
2. Check code examples di documentation
